Bài 64: Ngôn ngữ học

Học từ vựng và khái niệm về ngôn ngữ học, cấu trúc ngôn ngữ, ngữ âm học và ngữ nghĩa học

🎯 Mục tiêu bài học:

  • Nắm vững từ vựng chuyên ngành ngôn ngữ học
  • Hiểu các nhánh chính của ngôn ngữ học
  • Phân biệt các khái niệm về ngữ âm, ngữ pháp và ngữ nghĩa
  • Áp dụng thuật ngữ ngôn ngữ học trong phân tích ngôn ngữ
